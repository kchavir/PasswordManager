<h1>Password Manager</h1>



<h2>Description</h2>
This is the program that allows user entering  or randomly generating website passwords and saving them to the "TXT" file.  <br />


<h2>Languages and Utilities Used</h2>

- <b>Python</b> 
- <b>TKInter</b>
- <b>Pyperclip</b>
